# 👕 E-Commerce Clothing Store (Mern Stack)

A modern React-based clothing e-commerce website UI where users can browse products, view categories, and manage cart visually.

## ✨ Features
- Men, Women, Kids sections
- Product cards with price & images
- Add to cart UI
- Category filtering
- Responsive design
- Clean modern layout

## 🛠 Tech Stack
- React.js
- CSS / Tailwind / Bootstrap
- JavaScript

## 🚀 Run Project Locally

npm install  
npm run dev
